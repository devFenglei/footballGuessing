伪类与伪元素

https://www.jianshu.com/p/2fdb26890e7d


伪类

伪类用于选择Dom之外的的信息，或是不能用简单选择器进行表示的信息。

前者包含那些匹配指定状态的元素，比如:visited :active

后者包含那些满足一定逻辑条件的Dom树中的元素，比如:first-child :first-of-type :target

常见的伪类有：

:avtive :hover :link :visited :first-child :focus :not


伪元素

伪元素为Dom树没有定义的虚拟元素。

不同于其他选择器，它不是以元素为最小选择单元，它选择的是元素指定的内容。比如::before表示选择元素内容之前的内容，也就是""，::selection表示选择元素被选中的内容

常见的伪元素有：

::first-letter ::first-line ::after ::before ::selection
